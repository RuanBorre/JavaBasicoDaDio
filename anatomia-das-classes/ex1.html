Anotações sobre: Anatomina das Classes - 1

A escrita de codigos de um programa é feita atraves da cmposição de palavras pé-definidas pela linguagem com as expressões que utilizamos pra determinar o nome do nossos arquivos para determinar o nome dos nossos arquivos, classes, atributos e métodos

É muito comum ser expressada na linguagem inglesa

  • O nome a classe deve ser de acordo com o nome do arquivo, por exemplo:
  • Se o nome do programa for FutebolBrasileiro, a classe devera seguir o mesmo padrão de nome.

  • Arquivo: futebolbrasileiro
    Classe: FutebolBrasileiro

Anatomia do código


public class MinhaClasse {

public static void main(String[] args) throws Exception {
System.out.println("Hello, World!")
}
}

Toda classe começará com letra maiúscula, juntamente com seu sucessores. Podemos abservar no código acima

main: metodo principal que ve seguir um padrao: public static void main(String []args)

void: só vai executar sem retornat mada

{} : metodo, todo corpo é representado por abre e fecha chaves.

args: argumentos ou parametros

System.out.print("") : imprimir um texto
