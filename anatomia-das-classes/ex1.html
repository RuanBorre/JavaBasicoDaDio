Padrão de Nomenclatura

Quando se trata de escrever códigos na linguagem java, é recomendado seguir algumas convenções na escrita. Por exemplo:

Todo arquivo ,java devera começar com letra maiuscula, por exemplo: MinhaClasse.java / Cliente.java /

Caso seja uma palavra composta a segunda palavra tambem deverá seguir essa regra. A classe devera ter o mesmo nome do arquivo, caso nao seja dara erro de escrita.

VARIAVEL

toda variavel deve ser escrita com letra minuscula, mas caso seja composta, a segunda palavra devra ser maiscula.

Para criar uma variavel que não seja alterada, usamos final, por exemplo:

Uma variavel chamada BR

final string BR = "Brasil"

Casos que a variavel não seja alterada, por exemplo: Valor de PI, quantidade de estados, 1x7 da copa.

Em outros casos como em datas, que muita das vezes o valor precise ser alterados, usamos int.

int ano = 2023

ano = 2024

Regras de Declaração de uma variavel

  • Deve conter apenas letras, _ , $ ou números de 0 a 9
  • Deve iniciar com uma letra maiúscula
  • Não pode iniciar com numeros, de preferencia que comece com letras.
  • Não pode conter espaço
  • Não pode usar palavras chaves da linguagem
  • O nome deve ser único dentro do escopo

Declarações inválidas de variaveis

  • int numero$um = 1; //$ não é permitido
  • int 1numero = 1 ; //Uma variável não deve começar com números/li>
  • int numero um = 1: //Uma variável não pode conter espaço
  • int long = 1; //não pode conter palavras reservadas (long)

Declarações válidas de variáveis

  • int numero$um = 1;
  • int numero1 = 1;
  • int numeroum = 1;
  • int longo = 1;

Anatomia de Classes - 6

Java Beans

Iniciativa conceitual para que possamos ter uma estruturação de escrita de como expressamos as nossas variaveis, metetodos, etc. Os nossos codigos precisam ser legiveis por qualquer um.

Como Aplicar?(variaveis)

  • Uma Variavel deve ser clara, sem abreviações ou definiçao sem sentido.
  • Uma variavel é sempre no singular, menos quando se referir a uma array ou coleção
  • Defina sempre uma unica idioma para suas variavel

Exemplos de erros:

  • double salMedio = 1450.50;
  • String emails = "aluno@escola.com"; (É apenas um gmail)
  • String mayName = "JOSH"; Defina apenas um idioma para o código.

Como Aplicar? (Métodos)

Os metodos devem ser nomeados como verbos, através de uma mistura de letras minúsculas e maiúsculas. Todas as letras que compõem o nome devem ser mantidas em minusculo. com execeção da primeira palavra composta a partir da segunda palavra .

Por exemplo:

  • somar (int n1, int n2){}
  • abrirConexao(){}
  • concluirProcessamento(){}
